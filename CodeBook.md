## CodeBook

This describes the variables, the data, and any transformations of up the data("Tidy.txt")

"Tidy.txt" is a space delimited file and the first row contains the header.

There are Two group of elements present in the dataset

1) ID Variable : The dataset is grouped by these variables
- Subject - identifier for the subject
- Activity Name - Type of activity being performed
